# Enterprise Technology Governance Framework (ETGF)

This repository is the authoritative engineering source for the Enterprise Technology Governance Framework.

## Baseline

All work packages SHALL conform to [`ETGF-BASELINE-1.0.md`](ETGF-BASELINE-1.0.md).

## Repository Structure

- `governance/` — policy, standards, principles, procedures, and models
- `registers/` — requirements, controls, regulatory traceability, and taxonomy registers
- `publication/` — Microsoft Word publication outputs and release packages
- `releases/` — release notes and version baselines

## Governing Rule

**Policy governs. Standards implement. Procedures execute. Engineering standards configure.**
