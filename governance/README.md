# Governance Artifacts

This directory contains governed source artifacts for the Enterprise Technology Governance Framework.

## Planned Structure

- `policy/` — Enterprise Technology Policy
- `standards/` — Enterprise Architecture, Technology Governance, and Technology Lifecycle Standards
- `principles/` — Enterprise Technology Principles
- `procedures/` — Operational governance procedures
- `models/` — ETGRM, EOM, ERM, and related governance models

All artifacts SHALL conform to the repository root baseline.
