# ETGF Controlled Templates

## Purpose

This directory contains reusable templates implementing ETGF policy, standards, controls, architecture governance, lifecycle governance, evidence management, and publication requirements.

## Initial Template Inventory

- Architecture Decision Record
- Architecture Review Submission
- Architecture Compliance Assessment
- Change Impact Assessment
- Exception Request
- Risk Acceptance Record
- Remediation Plan
- Control Test Result
- Evidence Submission Record
- Technology Lifecycle Record
- Technology Retirement Plan
- Reference Architecture
- Architecture Pattern
- Service and Capability Mapping
- Vendor Technology Assessment
- Operational Resilience Assessment
- Procedure Traceability Record
- Publication Readiness Checklist

## Template Control Rules

1. Templates shall reference stable ETGF identifiers.
2. Mandatory fields shall not be removed in local derivatives.
3. Tool-specific versions shall preserve the canonical data model.
4. Approved templates shall carry version, owner, lifecycle status, and source reference.
5. Superseded templates shall remain available for record-history purposes and be clearly marked as superseded.
