# ETGF v1.0 RC1 Roadmap

## Objective

Publish the Enterprise Technology Governance Framework as seven authoritative source-controlled volumes with stable identifiers, master registers, Word publication derivatives, and release controls.

## Frozen Direction

- GitHub is the authoritative source repository.
- Markdown and structured registers are the working baseline.
- Microsoft Word is the controlled publication derivative.
- Policy governs. Standards implement. Procedures execute.
- Governance bodies, decision rights, and architecture remain frozen.
- ServiceNow EA/APM/SPM/CMDB/CSDM 5.0 provides technology-management integration.
- Archer remains the authoritative GRC platform for risk, controls, exceptions, assessments, and evidence.

## Release Gates

1. Architecture freeze confirmed.
2. Policy baseline consolidated.
3. Standards baseline consolidated.
4. Governance engineering artifacts normalized.
5. Repository object and relationship models reconciled.
6. Master registers generated.
7. Cross-references validated.
8. Word derivatives generated.
9. Approval placeholders inserted.
10. RC1 manifest and release notes completed.

## Definition of Complete

RC1 is complete when all authoritative source files, master registers, Word derivatives, identifiers, traceability links, release metadata, and acceptance checks are reconciled and publication-ready.
