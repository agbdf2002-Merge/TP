# Repository Governance

## Objective

This repository is the source-controlled working library for the Enterprise Technology Governance Framework.

## Source of Truth

Markdown files are the working source. Word documents are formal publication outputs. Registers are authoritative governance indexes.

## Review Model

Material changes should be reviewed by Office of the CIO, Enterprise Architecture, Technology Risk, Information Security, Legal/Compliance, and Internal Audit as applicable.

## Commit Standard

Use structured commit messages:

```text
<artifact>: <change summary>
```
