# Enterprise Technology Taxonomy Seed

## Purpose

The Enterprise Technology Taxonomy provides the common language for policy, standards, controls, evidence, ServiceNow EA/APM, CSDM, CMDB, Archer/IRM, engineering delivery, and audit traceability.

## Technology Domains

| Domain ID | Domain | Description | Executive Owner | Responsible Owner |
|---|---|---|---|---|
| TD-001 | Enterprise Architecture | Architecture strategy, standards, patterns, governance, and decisioning | CIO | Head of Enterprise Architecture |
| TD-002 | Application Architecture | Application portfolio, lifecycle, patterns, rationalization, and modernization | CIO | Application Architecture |
| TD-003 | Data Architecture | Data domains, integration, classification, analytics, and governance alignment | CDAO / CIO | Data Architecture |
| TD-004 | Security Architecture | Secure-by-design, IAM, encryption, threat modeling, and risk alignment | CISO | Security Architecture |
| TD-005 | Cloud Platform | Cloud landing zones, controls, cost, resilience, and platform patterns | CIO | Cloud / Platform Architecture |
| TD-006 | Infrastructure Architecture | Compute, storage, network, facilities, and hosting patterns | CIO | Infrastructure Architecture |
| TD-007 | API and Integration | APIs, events, services, integration patterns, and developer experience | CIO | Integration Architecture |
| TD-008 | Operational Resilience | Availability, recovery, continuity, observability, and incident readiness | COO / CIO | Technology Operations |
| TD-009 | Third-Party Technology | Vendor, SaaS, managed services, external products, and concentration risk | CIO / CRO | TPRM / Architecture |
| TD-010 | AI and Emerging Technology | AI enablement, experimentation, model risk, emerging technology, and innovation governance | CIO / CDAO | EA / AI Governance |

## Minimum Taxonomy Metadata

Every taxonomy object shall define ID, name, description, owner, executive owner, lifecycle state, related policy domain, related standards, related requirements, related controls, system of record, and review cadence.
