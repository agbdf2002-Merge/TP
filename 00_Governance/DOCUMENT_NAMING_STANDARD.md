# Document Naming Standard

## Naming Convention

```text
<ID>_<Document_Name>_v<version>.<extension>
```

## Examples

```text
ETP-001_Enterprise_Technology_Policy_v0.5.md
ETS-001_Enterprise_Architecture_Standard_v0.1.md
ARB-001_Architecture_Review_Board_Charter_v0.1.md
RTM-001_Requirements_Traceability_Matrix_v0.1.xlsx
```

## Identifier Prefixes

| Prefix | Artifact |
|---|---|
| ETP | Enterprise Technology Policy |
| ETS | Enterprise Technology Standard |
| ETGF | Enterprise Technology Governance Framework |
| RTM | Requirements Traceability Matrix |
| CTRL | Control |
| CO | Control Objective |
| ADR | Architecture Decision Record |
| EX | Exception |
| KPI | Key Performance Indicator |
| KRI | Key Risk Indicator |
