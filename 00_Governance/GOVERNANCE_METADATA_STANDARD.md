# Enterprise Governance Metadata Standard

## Purpose

Define required metadata for governance artifacts.

## Required Metadata

| Field | Requirement |
|---|---|
| Document ID | Unique identifier required |
| Document Type | Policy, Standard, Charter, Framework, Register, Procedure, Appendix |
| Owner | Accountable executive or role |
| Responsible Organization | Organization responsible for maintenance |
| Approval Authority | Committee or executive approver |
| Version | Semantic or policy version |
| Status | Draft, Review, Approved, Published, Retired |
| Review Frequency | Annual minimum unless otherwise defined |
| Effective Date | Required before publication |
| Related Requirements | ETP-RQ identifiers |
| Related Controls | CTRL identifiers |
| Regulatory Mapping | FFIEC, FDIC, OCC, Federal Reserve, NIST, CIS, COBIT, ITIL, ISO |
| System of Record | ServiceNow, Archer, ADO, Jira, etc. |
