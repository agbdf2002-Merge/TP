## Purpose

This document establishes the immutable architectural baseline for the Enterprise Technology Governance Framework (ETGF). It defines the approved governance architecture, operating model, editorial standards, object model, governance hierarchy, naming conventions, and production rules governing all subsequent artifacts.

6. **Tier 4:** Engineering Standards, Reference Architectures, Runbooks, Implementation Guides

## Governance Rule

**Policy governs. Standards implement. Procedures execute. Engineering Standards configure.**

## Editorial Standard

- Use RFC 2119 terminology.
- Mandatory policy statements begin with **“The Bank shall…”**
- Policies contain governance only.
- Standards define implementation requirements.
- Procedures define operational execution.
